This is a startpage for my Safari.

I might add more features in the future, depending on my personal needs. But for now, this works fine.
# Features
- Codeforces tracker: When I hit "Enter coding mode", a timer starts. When I exit coding mode, the ASCII wave moves up depending on the number of minutes that I coded for.
- Bookmarks: Bookmarks to my frequently visited pages

The coding timer is stored in LocalStorage and automatically resets to 0 each day.
